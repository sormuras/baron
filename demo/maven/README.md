# Maven Demo

[maven-archetype-quickstart](https://maven.apache.org/archetypes/maven-archetype-quickstart)

```text
project
|-- pom.xml
`-- src
    |-- main
    |   `-- java
    |       |-- $package
    |       |   `-- App.java
    |       `-- module-info.java
    `-- test
        `-- java
            |-- $package
            |   `-- AppTest.java
            `-- module-info.[java|test]
```
